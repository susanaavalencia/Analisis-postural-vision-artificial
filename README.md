##  Descripción del Dataset RAW

El conjunto de datos incluye:
- 4 ejercicios:
  - Sentadilla
  - Peso muerto
  - Curl de bíceps
  - Press de banca
- Tomas desde dos ángulos:
  - Frontal
  - Lateral
- 2 semanas de grabación.
- Formato de video: `.mp4`.

---

##  Propósito
Estos videos sirven como entrada para:
- el preprocesamiento de video,
- la estimación de pose,
- la generación del dataset procesado.
